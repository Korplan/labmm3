INSTRUÇÕES PARA O USO DOS JOGOS


NÚMEROS

Neste jogo é apresentado um sumo composto por um conjunto de frutas e/ou legumes. O jogador terá que escolher a combinação de frutas/legumes correnta de entre as opções apresentadas. Caso acerte passa para a fase seguinte, caso erre terá que tentar novamente.


PALAVRAS

Neste jogo é apresentada uma imagem de uma fruta ou legume. Neste caso, o jogador deverá completar a palavra, que se encontra dividida em sílabas. Para tal deverá escolher uma das síladas disponíveis. Caso acerte passa para a fase seguinte, caso erre terá que tentar novamente.


MEMÓRIA

O desafio neste jogo consiste em encontrar os pares escondidos nas cartas. Cada vez que o jogador selecione uma carta ela irá virar-se e mostrar a fruta ou legume. Quando o par for encontrado, essas cartas ficarão visíveis, mas quando errar as cartas voltam a vira-se ocultando a fruta ou legume. Quando o jogador conseguir descobrir todos os pares passa para a fase seguinte.


CORES

No jogo das cores é apresentada a imagem de uma fruta ou legume. O jogador deverá selecionar uma ou duas cores primárias por forma a obter a cor correta da fruta ou legume apresentada. Após selecionar a(s) cor(es) necessárias, terá que selecionar a cor por forma a pintar a fruta ou legume. Caso acerte passa para a fase seguinte, caso erre terá que tentar novamente.



Em cada um dos jogos estará disponível no canto superior esquerdo um ícone de acesso à ajuda help.

AJUDA


No menu de ajuda são disponibilizadas duas opções:
1. Obter ajuda, eliminando uma das alternativas erradas - através do botão "Obter Ajuda";
2. Obter a resposta correta - através do botão "Obter Solução".


Nota que se for selecionada a opção "Obter Solução" o jogador não passará ao nível seguinte de dificuldade.
